popy.utils.create_agent_graph(agents, node_attrs=None, include_0_weights=True)[source]

Create a Graph from a model’s agent list.

Parameters:

  • agents (AgentList) – A model’s agent list

  • node_attrs (Optional[list]) – A list of agent attributes

  • include_0_weights (bool) – Should edges with weight 0 be displayed?

Return type:

Graph

Returns:

A weighted graph created from a model’s agent list. Agents are connected if they are neighbors in the model. Their connecting edge include the contact_weight as “weight” attribute.

popy.utils.create_contact_matrix(agents, attr='id', weighted=False, plot=True, annot=False, return_df=False)[source]

Create a contact matrix as a DataFrame from a given model’s agent list.

Parameters:

  • agents (list | AgentList) – A list of agents.

  • attr (str) – The agent attribute which is shown in the matrix.

  • weighted (bool) – Should the contacts be weighted? Defaults to False.

  • plot (bool) – Should the matrix be plotted? Defaults to False.

  • annot (bool) – Should the plottet matrix be annotated? Defaults to False.

Return type:

DataFrame

Returns:

A DataFrame containing a contact matrix based on attr.
